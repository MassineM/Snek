# Snek
Snek game online
-----------------------------------------------------

How to compile:

1. Install Node modules:
  Execute the following command in API + APP folders:
    npm install
    
2. Run API and APP servers:
  Execute the following command in API + APP folders:
    npm run dev

3. Run WebGL server (Unity launcher):
  Execute following command in API/WebGL folder:
    node server.js
    
Have fun!
